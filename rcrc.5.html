NAME

rcrc — configuration for rcm

SYNOPSIS

X=“a b c”

DESCRIPTION

The rcm dotfile manager can be configured using a .rcrc file in your home directory. The format is POSIX shell, and it is sourced in by the lsrc(1), mkrc(1), and rcup(1) programs.

It supports these variables:

TAGS
the default tags used by rcup
DOTFILES_DIRS
the source directories for dotfiles. The first in the list is the canonical source. The default value is ~/.dotfiles

FILES

~/.dotfiles ~/.rcrc

EXAMPLES

TAGS="freebsd development email git laptop gmail notmuch"
DOTFILES_DIRS="/home/mike/.dotfiles /usr/share/dotfiles"

SEE ALSO

lsrc(1), mkrc(1), rcdn(1), rcup(1), rcm(7)
